Visual Studio

Visual Studio Code is a lightweight but powerful source code editor which runs on your desktop and is available for Windows, macOS and Linux. It comes with built-in support for JavaScript, TypeScript and Node.js and has a rich ecosystem of extensions for other coding languages.

The Console Screen

The Developer Console is a special screen in The "Inspect" Menu. To Open it right-click on the page and select "Inspect", and then click Console. It is mainly used for debugging as it make the JavaScript written print on the console.
